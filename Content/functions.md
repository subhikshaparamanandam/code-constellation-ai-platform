# Python Functions


## What is a Function?

A function is a block of code that performs a specific task.
Think of a function like a machine: **Input → Process → Output**

## Why Use Functions?
* Avoid repeating code
* Improve readability
* Make debugging easier

## Example Function
```python
def greet():
    print("Welcome to Code Constellation")

greet()
```

### Function with Parameters
```python
def add(a, b):
    return a + b

result = add(5, 3)
print(result)
```
