# Python Data Types


## What is a Data Type?

A data type tells Python what kind of value a variable is storing.

Just like:
* Water bottle stores water
* Pen stores ink
* Python uses data types to understand how to handle values.

## Common Data Types in Python

| Data Type | Example | Meaning |
| :--- | :--- | :--- |
| `int` | `10` | Whole numbers |
| `float` | `3.14` | Decimal numbers |
| `str` | `"Hello"` | Text |
| `bool` | `True` | True or False |

## Example in Python
```python
age = 20          # int
height = 5.6      # float
name = "Ravi"     # str
is_passed = True  # bool
```

Python automatically detects the data type.

### Check Data Type
```python
print(type(age))
print(type(name))
```
